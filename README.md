# SolidBooked Pro Setter Funnel

Static GitHub Pages package for `setter.solidbookedpro.com`.

## Current Launch Mode

- `index.html` is self-contained.
- Missing founder-video media is allowed for the first publish.
- Missing Vapi assistant config falls back to simulated practice calls.
- Missing backend submit endpoints fall back to client-side qualification/local storage.

Replace the `APP_CONFIG` values in `index.html` when the final video, call audio, Vapi assistants, calendar, and applicant API are ready.
